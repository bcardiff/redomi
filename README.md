# redomi

**Experiment** in Remote Dom Manipulation.
Build an app in crystal using the browser as a rendering engine.

Check (sample/basic.cr)[sample/basic.cr]

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  redomi:
    github: bcardiff/redomi
```
